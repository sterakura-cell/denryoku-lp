# Google Apps Script 連携手順（フォーム→スプレッドシート）

フォームの送信内容を **Googleスプレッドシート（＝管理シート）に自動追記**し、
**明細ファイルをGoogleドライブに保存**するための設定です。完全無料・送信数の制限なし。

所要：**15〜20分**。むずかしい操作はありません。

---

## STEP 1. スプレッドシートを作る

1. https://sheets.google.com で **空白のスプレッドシート** を新規作成。
2. 名前を「動力電気代 問い合わせ管理」などに（任意）。
   - 列の見出しは **自動で作られる** ので、手入力は不要です。

## STEP 2. スクリプトを貼り付ける

1. そのスプレッドシートのメニュー **拡張機能 → Apps Script** を開く。
2. 最初からある `function myFunction() {}` を**すべて削除**。
3. このフォルダの **`apps-script/Code.gs` の中身を全部コピーして貼り付け**。
4. （任意）新着をメールで受け取りたい場合、先頭の
   ```js
   var NOTIFY_EMAIL = "";
   ```
   を、受け取りたいアドレスに変更：
   ```js
   var NOTIFY_EMAIL = "あなた@example.com";
   ```
5. 💾 保存（プロジェクト名は「動力電気代LP受信」など任意）。

## STEP 3. ウェブアプリとしてデプロイ

1. 右上の **デプロイ → 新しいデプロイ**。
2. 歯車（種類の選択）→ **ウェブアプリ** を選択。
3. 設定：
   - 説明：任意
   - **次のユーザーとして実行：自分**
   - **アクセスできるユーザー：全員（Anyone）** ←重要
4. **デプロイ** をクリック。
5. 初回は「承認」を求められます → 自分のGoogleアカウントで承認
   （「このアプリは確認されていません」と出たら **詳細 → （プロジェクト名）に移動 → 許可**）。
6. 表示される **ウェブアプリURL（`https://script.google.com/macros/s/XXXX/exec`）をコピー**。

> 動作確認：このURLをブラウザで開いて「受信エンドポイントは稼働中です。」と出ればOK。

## STEP 4. URLを教えてください

コピーした **`.../exec` のURLを共有**してください。
こちらで `script.js` の `GAS_ENDPOINT` に設定し、再デプロイ（GitHubへpush）します。

設定後はフォーム送信すると：
- スプレッドシートに1行自動追記（受付日・会社名・ランク・年間電気代・15%/20%削減見込み・対応状況「未対応」など）
- 明細ファイルはドライブの「動力電気代_明細」フォルダに保存され、シートにリンクが入る
- （メール設定時）新着が件名「【動力電気代診断】◯◯様／月額◯円／ランクS」で届く

## よくある質問

**Q. 列は自分で作るの？**
いいえ。初回の送信時に見出し行が自動作成されます。

**Q. スクリプトを直したら？**
Apps Scriptで編集後、**デプロイ → デプロイを管理 → 編集（鉛筆）→ バージョン「新バージョン」→ デプロイ**で反映。
URLは変わりません。

**Q. 明細ファイルが大きいと？**
写真・PDFなら通常問題ありません。極端に大きいファイル（数十MB）は避けてもらう運用が安全です。

**Q. 個人情報は大丈夫？**
スプレッドシート・ドライブともにあなたのGoogleアカウント内に保存されます。
共有リンクは「リンクを知っている人が閲覧可」になるため、シート自体は**第三者に共有しない**でください。
